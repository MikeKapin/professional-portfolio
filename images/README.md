## How to Add Photos

1. Save each photo with the filenames above
2. Use JPG format for best web compatibility
3. Optimize images (compress to < 500KB each)
4. The HTML is already configured to use these paths

## Image Optimization Tools

- **Online:** TinyJPG.com, Squoosh.app
- **Desktop:** IrfanView, GIMP
- **Command Line:** `convert image.jpg -quality 85 -resize 1920x1080 optimized.jpg`
